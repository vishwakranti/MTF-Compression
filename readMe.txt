
Instructions to compile and execute the programs are below:

The source code is located with the "src" directory.

The project was created using Eclipse IDE.

After compiling the .java files in the "src" folder, the program can be run like below from
the folder which contains the class files

1. To encode the input text file type the following in the terminal

java MTFencoder inputTextFile > EncodedFile


2. To decode the file

java MTFdecode EncodedFile
